Runtime Dependencies

This is a hybrid build. The core app runs locally from file://, but Square spreadsheet import/export and ZIP image downloads rely on two runtime CDN libraries: XLSX and JSZip. If those libraries do not load, core local inventory data still works, but spreadsheet and ZIP features will be unavailable.

XLSX — required for Square spreadsheet import/export and spreadsheet-based sales parsing.
JSZip — required for bulk image ZIP downloads.
